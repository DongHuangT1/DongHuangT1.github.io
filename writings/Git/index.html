基本命令

git init                        #初始仓库
git status                      #显示状态
git add .                       #添加暂存
git commit -m "***"             #提交仓库
git log --oneline               #查看提交
git reflog                      #历史版本
git reset --hard "***"          #回退版本
git pull                        #拉取代码
git push -f                     #推送代码
------------------------------------------------
git branch [Name]               #查看分支
git checkout Name               #切换分支
git merge Name                  #合并分支
------------------------------------------------
git remote add origin git@github.com:User/Repo.git
git push -u origin master       #上传GitHub

克隆加速

由于天朝网络原因，导致Git克隆异常缓慢，因此需要我们另辟途径进行加速

fastgit

git clone https://hub.fastgit.xyz/User/Repo.git

gitclone

git clone https://gitclone.com/github.com/User/Repo.git

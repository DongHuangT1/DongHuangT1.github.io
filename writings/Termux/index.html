简介

Termux是一个 Android终端仿真器 和 Linux环境应用程序 ，无需root或设置。 自动安装了最小的系统，可以使用APT软件包管理器安装其它软件程序。

  • GitHub
  • APK下载
  • Google Play

快速上手

基本命令

pkg search <query>              #搜索软件包
pkg install <packages>          #安装软件包
pkg uninstall <packages>        #卸载软件包
pkg show <packages>             #显示包信息
pkg reinstall <packages>        #重新安装包
pkg upgrade                     #升级软件包
pkg list-all                    #列出所有包
pkg files <package>             #显示包路径
pkg list-installed              #列出已安装

国内镜像

echo "deb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main" > $PREFIX/etc/apt/sources.list
echo "deb https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable"  > $PREFIX/etc/apt/sources.list.d/game.list
echo "deb https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science stable" > $PREFIX/etc/apt/sources.list.d/science.list
apt update && apt upgrade -y

基本工具

pkg install git vim php curl wget tree unzip golang python openssh -y
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
go env -w GOPATH=$PREFIX && go env -w GOPROXY=https://goproxy.io,direct

美化终端

sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"

重新选择色彩样式: chcolor 或 ~/.termux/colors.sh

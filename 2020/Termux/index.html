简介

Termux是一个 Android终端仿真器 和 Linux环境应用程序 ，无需root或设置。 自动安装了最小的系统，可以使用APT软件包管理器安装其它软件程序。

  • GitHub
  • APK下载
  • Google Play

快速上手

基本命令

pkg search <query>              #搜索包
pkg install <packages>          #安装包
pkg uninstall <packages>        #卸载包
pkg show <packages>             #显示包信息
pkg reinstall <packages>        #重新安装包
pkg upgrade                     #升级软件包
pkg list-all                    #列出所有包
pkg files <package>             #显示包路径
pkg list-installed              #列出已安装包

国内镜像

sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list
sed -i 's@^\(deb.*games stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable@' $PREFIX/etc/apt/sources.list.d/game.list
sed -i 's@^\(deb.*science stable\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science stable@' $PREFIX/etc/apt/sources.list.d/science.list
apt update && apt upgrade

基本工具

pkg install vim curl wget git unzip unrar tree openssh python

美化终端

sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"
Enter a number, leave blank to not to change: 14    #选择色彩样式
Enter a number, leave blank to not to change: 6     #选择字体样式

重新选择色彩样式: chcolor或~/.termux/colors.sh
重新选择色彩样式: chfont或~/.termux/fonts.sh

键盘优化

mkdir $HOME/.termux        #如果存在请忽略
echo "extra-keys = [['ESC','/','-','HOME','UP','END','PGUP'],['TAB','CTRL','ALT','LEFT','DOWN','RIGHT','PGDN']]" > $HOME/.termux/termux.properties

安全工具

Nmap

网络扫描和主机检测工具

pkg install nmap

Dirb

基于字典的Web目录扫描工具

pkg install dirb

Commix

自动化检测系统命令注入工具 项目地址

git clone https://github.com/commixproject/commix.git
#国内 git clone https://gitee.com/DongHuangT1/commix.git
echo "alias commix='python ~/commix/commix.py'" >> ~/.zshrc;source ~/.zshrc

Sqlmap

自动化检测，利用SQL注入漏洞 项目地址

mkdir ~/.pip;echo -e "[global]\nindex-url = https://pypi.tuna.tsinghua.edu.cn/simple" > ~/.pip/pip.conf 
pip install sqlmap
